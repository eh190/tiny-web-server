# Simple Web Server in C

An implementation of the **Tiny Web Server** described in *Computer Systems: A Programmer's Perspective*.

## Overview
This project demonstrates a minimal HTTP server written in C.  
It handles basic GET requests, parses HTTP headers, and can be extended to serve static or dynamic files
